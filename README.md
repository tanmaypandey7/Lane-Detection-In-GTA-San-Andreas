# Self Driving Car In GTA San Andreas

## How To Run
1. Install [DxWnd](https://sourceforge.net/projects/dxwnd/)
2. Drag and drop gta_sa.exe from the installation directory to DxWnd.
3. Launch the program from DxWnd.
